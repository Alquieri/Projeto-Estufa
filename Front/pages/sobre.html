Sobre o Projeto

O Projeto Estufa Inteligente É uma plataforma de monitoramento agrícola que utiliza sensores físicos para acompanhar parâmetros ambientais como temperatura e umidade em tempo real. Nosso sistema conta com um back-end em Node.js e uma interface web simples e acessível, ideal para agricultores que buscam otimizar o cuidado com suas plantações.
